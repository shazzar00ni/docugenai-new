# Quick Start Guide

Welcome to **DocuGenAI**! This guide will help you create your first documentation website in minutes.

## Step 1: Prepare Your Markdown Files

Make sure your documentation is in Markdown format (`.md` or `.markdown` files). Here's a simple structure:

```markdown
# Project Title

## Introduction

Your introduction here...

## Installation

Step-by-step installation guide...

## Usage

How to use your project...
```

## Step 2: Upload Your Files

1. Open DocuGenAI in your browser
2. Scroll to the "Upload Your Documentation" section
3. Either:
   - **Drag and drop** your `.md` files into the upload area
   - **Click** the upload area to browse and select files

## Step 3: Review Your Files

After uploading, you'll see a list of your files with:

- File name
- File size
- Option to remove files

You can upload multiple files at once!

## Step 4: Generate Your Website

1. Click the **"Generate Website"** button
2. Wait a few seconds while AI processes your content
3. The AI will:
   - Parse your markdown
   - Extract headings and structure
   - Create navigation
   - Generate a beautiful website

## Step 5: Preview and Download

Once generated, you'll see:

- **Live Preview** - See your website in action
- **Edit Button** - Go back to add/remove files
- **Download Button** - Download your complete website as HTML

The AI automatically creates navigation based on your file names and heading structure.

Your generated website works perfectly on:

## Troubleshooting

### Files Not Uploading?

- Make sure files have `.md` or `.markdown` extension
- Check file size (should be reasonable)
- Try uploading one file at a time

### Website Not Generating?

- Ensure at least one file is uploaded
- Check browser console for errors
- Try refreshing the page

### Preview Not Showing?

- Wait for generation to complete
- Check if popup blockers are enabled
- Try a different browser
